# Composite pattern
- 객체들을 트리 구조로 구성하여 부분과 전체를 나타내는 계층 구조로 만들 수 있음
- 클라이언트에서 개별 객체와 다른 객체들로 구성된 복합 객체를 똑같은 방법으로 다룰 수 있음
