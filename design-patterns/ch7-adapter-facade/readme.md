# Adapter pattern
- 한 클래스의 인터페이스를 클라이언트에서 사용하고자하는 다른 인터페이스로 변환
- 어댑터를 이용하면 인퍼페이스 호환성 문제를 해결

## 객체 어댑터
- 구성(composition)을 사용

## 클래스 어댑터
- 상속을 사용
- 다중 상속이 이용한 경우 사용
